Q: What is a URL and how is it related to URIs?

A reference to a resource that specifies the location of the resource on a computer network and a mechanism for retrieving it. A URL is a specific type of URI.

Q: Name as many safe HTTP methods as you can.

A safe method has no effect on the server. Safe methods are GET, HEAD, OPTIONS, and TRACE. They are read-only, but they do not exclude side effects like charging an advertising account.

Q: How do HTTP requests work?
